Syntax Help
  • Basic

    Basic Calls can be added by simply entering the text to be displayed on the card without any fancy syntax stuff. These Calls are automatically limited to one response per player. Calls without any underscores will not appear multiple times after completing a round.

    Examples:

    What made Spock cry?
  • Underscores

    Instead of appending a player's response to the end of the call's text, underscores can be placed anywhere to complete a call's text. With more underscores, the players need to pick more cards to participate in a round.

    Examples:

    You know, once you get past _, _ ain't so bad.
  • Composite Formatting

    Composite Formatting allows you to reuse text in multiple places. The syntax is similar to C#, and can be used as many time as you want. This can also be combined with underscores, though you cannot reuse the text from those.

    Examples:

    He said that {0} on {1} is better than {0} without {1}
  • HTML Support

    Calls also support HTML, so you can include images, colored text or even unnecessary animations.

    Examples:

    This is a call that renders <span style="text-transform: uppercase">_</span> in uppercase
